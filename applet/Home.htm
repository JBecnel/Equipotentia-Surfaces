Brief Description

The Equipotential Surfaces Applet calculates loci of points with the same potential. These loci are called equipotential surfaces. This applet allows any number particles to be placed in a electric field of any size; it can then find the potential at any given point and the equipotential suface that this point is a member of. The applet also can be used to search a field to find a point with a given potential.

The applet is written using JDK 1.2 (which requires the Java Plug-in) and was primarily tested on a Windows 98 machine using Internet Explorer. The applet presents a visual tool for studying the shapes and properties of equipotential sufaces. See the links below to run the applet or view the documentation or source code for the applet.
